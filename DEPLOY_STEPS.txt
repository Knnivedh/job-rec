Build fails?
→ Check the deploy log in Netlify for errors
→ Usually missing environment variables

App shows errors?
→ Make sure both API keys are added correctly
→ No spaces or quotes around the keys
→ Redeploy after adding keys

Jobs don't load?
→ RapidAPI key might be invalid
→ Check your RapidAPI account is active

AI doesn't work?
→ NVIDIA API key might be invalid
→ Check your NVIDIA account is active
